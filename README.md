# Welcome to Inventory System

**About The Project**

The inventory System is the program used to manage and track stock items in a store or warehouse. It allows users to add, update, search, monitor stock levels, record new products. The system helps ensure accurate stock control, reduces manual errors, and improves the efficiency of inventory management.

It starts by asking the user to choose what he will like to do whether:
1. View all shoes 
2. Capture a new shoe
3. Re-stock a shoe
4. Search for a shoe
5. Calculate value per item
6. View shoe with highest quantity
7. Exit

If the user choose no:1  - will iterate over the shoes list and print the details of the shoes
If the user choose no:2  - program will allow a user to capture data  about a new shoe
If the user choose no:3  - program will find the shoe with the lowest quantity, which is the shoes that need to be re-stocked. Then, it will ask                             the user if they want to add this quantity of shoes and then update it.
If the user choose no:4  - this option will search for a shoe from the list using the shoe code and return data to be printed
If the user choose no:5  - this will calculate the total value for each shoe item
If the user choose no:6  - this will get the Product with the highest quantity and print it as being for sale
If the user choose no:7  - This will exit the prgram
